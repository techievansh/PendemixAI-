# Streamlit Project

This project is a Streamlit web application that allows users to interact with data and view results through an easy-to-use web interface. The application uses Python libraries such as Streamlit, Pandas, NumPy, and Scikit-learn to process data and display outputs or predictions. 

## How to Run
Install dependencies:
pip install -r requirements.txt

Run the app:
streamlit run app.py

## Project Files
app.py – main application  
requirements.txt – dependencies
